# TDD Practice
TDD 연습을 위한 Repository

## 실행방법
```
$ ./gradlew clean test
```

## TDD 주기
1. 작은 테스트를 하나 추가한다.
1. 모든 테스트를 실행해서 테스트가 실패하는 것을 확인한다.
1. 조금 수정한다.
1. 모든 테스트를 실행해서 테스트가 성공하는 것을 확인한다.
1. 중복을 제거하기 위해 **리팩토링**을 한다.
